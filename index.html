HealthMate (2017)

  • Project Overview: This was the final year project of my bachelor's degree. User can book a doctor's appointment using an application as well as a website. Also, they can check the doctor has arrived at a clinic/hospital or not. Furthermore, they can also see how many patients are waiting in a queue and they will get estimated time through SMS and email notification. The motto behind this project is sometimes the patient has to wait in a critical situation at hospital/clinic. So, If the patient can check the approximate consult wait time then he/she can plan to leave home accordingly and get rest at home.

  • Four modules in this project: Patient, Admin, Doctor and Receptionist.
  • Admin can manage all doctor, receptionist and patient profiles. Also, they can check appointment details.
  • Used java hibernate framework to create web API. So, I did not have to write same business logic for android and web application.
  • Implemented python script to send notification to patients regarding their consultation time.
  • Used way2sms and Gmail services to send notification.
  • Technology & Services: Java, Hibernate, Python, HTML, CSS, Ajax, Microsoft SQL server, Way2SMS, Gmail
