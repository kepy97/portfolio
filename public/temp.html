Actual Problem: EXIF

EXIF is a format in which pictures meta data can be found. These meta data may include information including device used for capturing like (iPhone 7, Samsung S4), camera specifications, GPS coordinates, orientation, etc,.

How typically devices take picture?

Typically if a device takes a picture it uses its sensors to rotate the image to the right orientation. Because of this wherever we view the photo we have the correct orientation for the image.

How iPhone takes picture?

iPhone, when takes the image it doesn’t rotate the image rather it keeps the image as it was taken and inserts the information with which the image can be correctly viewed. The is something like 90 degree clockwise. It expects the software used for viewing to rotate the image taken to correct orientation.

Why does it do so?

hmm… maybe to save battery in iphone, or maybe it will cause a sight delay to save the image.
