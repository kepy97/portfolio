Hide option to Publish to SharePoint in MS Access

Microsoft has been binding Office to Skydrive/Onedrive/SharePoint more and more since 2013. For those of us who don't use the cloud, this integration is more of a headache than a beneficial feature.

The Problem

Whenever user open the Microsoft Access file, the yellow alert bar that says "Save to SharePoint Site" is showing up. However, user don't want to do it but it pops up again and again on starting. Many people are posting about this problem, even I found a post that posted in Microsoft Community page on December 7, 2012. and I found only one reply that posted after 3 years from another user. The user commented: "Sad that nobody at MS can answer this question, even to say "no, it is not possible". Check out article Access 2010: Remove "Save to SharePoint Site" yellow bar

Solution

The below solution can work on all of these Access version: Access 2003 (version 11.5), Access 2007 (version 12.0), Access 2010 (version 14.0), and Access 2013 (version 15.0). (source: wiki)

Steps:

  • Create the new form in MS Access (From Menu bar-> Select Create -> Click on Form)
  • Save that form as "RemoveAlert" and open "Design View" of that form
  • Right click anywhere in "RemoveAlert" form and click "Form Properties"
  • Toolbar will appear on right hand side. Toolbar -> Event -> On Load (Click on three dots appear in right hand side)
  • Choose "Code Builder" from the menu
  • Then it will open Microsoft Visual Basic and paste this line "CurrentDb.Properties.Delete" and Save
