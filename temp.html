Here's the common problems that you might face in react-native

I hope you have just started to learning mobile app development using react native. There are lots of challenges to adopting the new technology. I will write about which issue took my much time to solve.

Synchronisation Problem

It's not easier to complete the installation of new technology. As you know, developers are keep trying to solve this problem. However, each solution creates the new problem. In react native, you can simply install any new libraries using the npm but there are lots of problem behind this. Developers keep introduces the new features, solve previous bugs, improving architecture etc.

In result, they keep update the versions of the libraries and we have to use latest version of the library. But, when you update only one library then it directly affects the other libraries that depended on it. You will find lots of issues about it in the Github.

Multiple commands produce

The common problem that everyone faces in the react native is Multiple commands produce. As I was working on mobile onboarding feature. So, I have to use react-native-vector-icons library to include icons in my project.

But, this libary includes fonts library that already comes with react native original libary as well as with other libraies (ex: react-native-base, galio-framework etc). So, when you build your project it clashes with other libray and leads to build fail.

Full Error

I almost spent one day to solve this problem. I have reinstalled all the libraries many times to catch the original problem. For that, I have created whole new project from the scratch and caught the issue. After that, I was looking for the solution. Many tutorials are said to unlink react-native-vector-icons library but still it's only a quick fix. I got many github and stackoverflow issue links but still no solution worked.

Finally, one article from gitmemory saved me. There solution worked for me. You can check whole post here. However, I'm attaching solution if you can't read the post.
